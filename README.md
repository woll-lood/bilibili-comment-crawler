# B站评论统一爬虫（优化版）

> 基于 [1dyer/bilibili-comment-crawler](https://github.com/1dyer/bilibili-comment-crawler) 深度优化，一个功能更强大、更稳定、更易用的B站评论爬虫工具。

## ✨ 优化亮点

相比原项目，本优化版在以下方面进行了全面升级：

| 优化项 | 原项目 | 优化版 |
| :--- | :--- | :--- |
| **内容类型** | 仅支持视频（BV号） | 支持视频（BV/**AV号**）、**番剧**、**动态** |
| **评论层级** | 仅爬取一级+二级评论 | **递归爬取所有层级**子评论，无遗漏 |
| **用户筛选** | 无法指定用户 | **支持按用户ID筛选**，可只爬取特定用户及其所有评论 |
| **输入方式** | 需手动修改代码中的BV号 | **命令行参数传入**，灵活方便 |
| **请求重试** | 无，请求失败直接崩溃 | **自动重试3次**，递增延迟 |
| **反爬延迟** | 固定0.5秒 | **随机延迟**（1-3秒）+ **阶梯冷却**（每500/1000条暂停10-60秒） |
| **数据字段** | 基础用户信息 | 新增 **UP主点赞/回复**、**当前账号是否点赞** 等互动字段 |
| **代码结构** | 重复代码多，视频/动态分脚本 | **单一脚本**，模块化函数，易于维护 |

## 🎯 功能特性

- **多内容类型支持**：视频（BV/AV号）、番剧、动态（opus）
- **多级评论爬取**：递归爬取所有层级的评论回复（不限于二级）
- **指定用户筛选**：只爬取特定用户及其参与的完整评论线索
- **丰富的用户信息**：用户等级、性别、IP属地、大会员状态、个性签名、头像
- **UP主互动标识**：UP主是否点赞或回复了该评论
- **当前账号互动**：当前登录账号是否点赞该评论
- **自动分页处理**：自动遍历所有评论页，无需手动干预
- **反爬策略优化**：WBI签名、随机延迟、自动重试、阶梯冷却
- **CSV导出**：标准CSV格式（UTF-8-BOM），兼容Excel和数据分析工具

## 📋 快速开始

### 1. 配置 Cookie

登录B站，按 `F12` 打开开发者工具 → `网络`(Network) 标签 → 刷新页面 → 找到任意请求中的 `Cookie` 值 → 复制完整内容。

在项目根目录创建 `bili_cookie.txt` 文件，将 Cookie 粘贴进去（**不要加引号或多余空格**）。
